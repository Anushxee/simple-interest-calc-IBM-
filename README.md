# Simple Interest Calculator

This project contains a Bash script to calculate simple interest based on user input.

## Features
- Calculates simple interest
- Takes principal amount, rate of interest, and time as input
- Displays the calculated simple interest

## Files
- `simple-interest.sh` - Bash script to calculate simple interest
- `LICENSE` - Apache 2.0 License
- `CODE_OF_CONDUCT.md`
- `CONTRIBUTING.md`

## Formula

Simple Interest = (Principal × Rate × Time) / 100
